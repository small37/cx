SleepGuard 项目介绍

macOS 菜单栏工具：防休眠、截图/选中文本识别、翻译展示、TouchBar 消息通道。

1. 项目目标

  • 阻止系统休眠（支持定时与合盖场景策略）。
  • 通过热键触发截图或读取选中文本，进行 OCR/翻译后弹窗展示。
  • 通过本地 Unix Socket 接收消息并渲染到 TouchBar（含图标、按钮、声音 DSL）。

3. 核心流程

A. 文本获取与翻译 
热键触发 -> AX/剪贴板读取选中文本 -> 弹窗显示原文 -> 百度翻译 -> 更新翻译区
B. 截图翻译 
热键触发截图 -> macOS 离线 OCR(Vision) -> 百度翻译 -> 双区弹窗展示
C. 防休眠 
菜单切换 -> IOPMAssertion + pmset disablesleep -> 定时模式倒计时 -> 到期自动恢复

4. 命令与 DSL

当前保留命令：tbmsg、tbdone。

tbmsg "[icon:hermes] [sound:start] 任务开始"
tbdone "[sound:end] 任务完成"

已支持关键词图标：claude / hermes / codex；声音模板：start / end。

5. 配置与权限

配置文件：~/.touchbar-island/config.json

  • 热键：hotkeys.text、hotkeys.screenshot
  • 翻译：baiduTranslate.appID、baiduTranslate.secret

权限要求：

  • 辅助功能（选中文本读取）
  • 屏幕录制（截图与 OCR）
  • 管理员权限（首次配置防休眠策略，建议 sudoers 白名单）

6. 构建运行

cd demo
./build.sh

输出并安装到：/Applications/SleepGuardDemo.app

当前版本已包含：防休眠、截图/选中文本翻译、TouchBar 图标与声音 DSL、倒计时菜单。

注意：保持 Bundle ID / 签名 / 安装路径稳定，可减少 macOS TCC 反复授权。
